## ⚡ Quick Start / 快速开始

```bash
# 1. Clone the repository / 克隆仓库
git clone https://github.com/binggan23/mimo-tts-studio.git
cd mimo-tts-studio/mimotts-app

# 2. Install dependencies / 安装依赖
npm install

# 3. Configure API key / 配置 API 密钥
# (A config.json will be auto-generated if missing)
# （如不存在，会自动生成 config.json）
```

<br/>

### 🔐 Configuration / 配置

Create `config.json` in the project root / 在项目根目录创建 `config.json`：

```json
{
  "TTSapikey": "your-mimo-api-key",
  "TTSurl": "https://api.xiaomimimo.com/v1/chat/completions",
  "TTSmodels": [
    "mimo-v2.5-tts",
    "mimo-v2.5-tts-voicedesign",
    "mimo-v2.5-tts-voiceclone"
  ]
}
```

<br/>

| Field / 字段 | Type / 类型 | Description / 说明 |
|:---|:---:|:---|
| `TTSapikey` | `string` | Your MiMo API key / MiMo API 密钥 |
| `TTSurl` | `string` | API endpoint (default provided) / API 端点 |
| `TTSmodels` | `string[]` | Available model IDs / 可用模型 ID 列表 |

> ⚠️ **Security / 安全提示:** `config.json` is gitignored. Never commit API keys to public repositories.<br/>
> `config.json` 已被 gitignore，切勿将 API 密钥提交到公开仓库。

<br/>

### 🚀 Run / 启动

```bash
npm run dev      # Development / 开发模式 → http://localhost:3000
npm run build    # Production build / 生产构建
npm run start    # Production server / 生产服务器
npm run lint     # Lint / 代码检查
```

## 🔌 API Reference / 接口文档

### `POST /api/tts` — Synthesize Speech / 语音合成

**Request / 请求:**

```json
{
  "model": "mimo-v2.5-tts",
  "text": "Hello, this is a test.",
  "voice": "Chloe",
  "style": "Warm and friendly tone",
  "tags": "[happy][gentle]"
}
```

| Parameter / 参数 | Required / 必填 | Description / 说明 |
|:---|:---:|:---|
| `model` | ✅ | `mimo-v2.5-tts` · `mimo-v2.5-tts-voicedesign` · `mimo-v2.5-tts-voiceclone` |
| `text` | ✅ | Text to synthesize / 待合成文本 |
| `voice` | ⚡ | Voice name (standard mode) / 音色名称 |
| `voiceDescription` | ⚡ | Voice description (design mode) / 音色描述 |
| `voiceAudio` | ⚡ | Base64 audio data URL (clone mode) / Base64 音频 |
| `style` | — | Style direction / 风格指令 |
| `tags` | — | Audio tags string / 音频标签 |

**Response / 响应:** WAV audio file (`audio/wav`, PCM16, 24kHz, mono)

> **Auto-segmentation / 自动分段:** Text exceeding 300 characters is automatically split at sentence boundaries and stitched with 200ms silence gaps.<br/>
> 超过 300 字符的文本会自动按句分割，并以 200ms 静音间隔拼接。

<br/>
